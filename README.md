# 企业文档分享平台

#### 介绍
基于知识分享理念而设计的文档管理分享与平台，解决文档分散、文档传递不可控等根本原因，帮助企业构建集中的海量文档管理平台，对文档进行全生命周期管理。


#### 安装教程

1. 安装nodejs
2. 数据库MongoDB
3. 网站服务serve
4. 部署文档分享应用
